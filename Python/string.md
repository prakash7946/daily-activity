## Strings in Python

A **string** in Python is a sequence of characters used to represent text. Strings are enclosed in **single quotes (' ')**, **double quotes (" ")**, or **triple quotes (''' ''' or """ """)**.

Strings are one of the most commonly used data types in Python and are used to store information such as names, messages, and sentences.

### Example
```python
name = "siva"
print(name)
